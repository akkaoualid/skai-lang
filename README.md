# oreview:
skai-lang is a hobby dynamicaly typed programming language made for scripting. [W.I.P]

## syntax:
### Hello, World:
```sk
print("Hello, World!");
```

### arithmeric operations:
```sk
print(3 - 2);  // 1
print(5 * 2); // 10
```
